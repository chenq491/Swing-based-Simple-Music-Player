# Swing-based Simple Music Player
Simple music player based on `javax.swing` Library

**Gif picture** and **lyrics** can be displayed

- You can view the music list and double-click to play any music. 
- You can switch the playback mode from random playback, sequential playback and single song cycle. 
- You can double-click the motion picture to switch pictures

The Gif file is placed in the `assets/gif` folder, the lyrics file is placed in the `assets/lyric` folder, and the song file is placed in the `assets/music` folder

Run the `src/com/chenq/main.java` file to start the project

**Only `.lrc` lyrics and `.wav` song files are supported**



基于`javax.swing`库的简单音乐播放器

展示**gif动图**和**歌词**

gif文件放入`assets/gif`文件夹，歌词文件放入`assets/lyric`文件夹，歌曲文件放入`assets/music`文件夹
